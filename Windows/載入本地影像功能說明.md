# 載入本地影像功能使用指南

## 功能概述

傷口自動化量測系統現在支援載入本地影像檔案進行分析，讓您可以使用預先拍攝的傷口影像進行量測和分析，無需即時攝影機。

## 支援的影像格式

系統支援以下影像格式：
- **JPEG** (.jpg, .jpeg)
- **PNG** (.png)
- **BMP** (.bmp)
- **TIFF** (.tiff, .tif)

## 使用步驟

### 1. 載入影像
1. 啟動傷口量測系統
2. 在左側面板點擊「載入本地影像」按鈕
3. 在檔案選擇對話框中選擇要分析的影像檔案
4. 系統會自動載入影像並顯示在原始影像區域

### 2. 處理影像
1. 載入影像後，點擊「處理載入影像」按鈕
2. 系統會執行以下處理流程：
   - 影像前處理（白平衡、雜訊抑制等）
   - AI 分類（傷口類型識別）
   - 量測計算（面積、周長、深度、體積）
3. 處理完成後，結果會顯示在右側面板

### 3. 查看結果
- **原始影像**: 顯示載入的原始影像
- **處理後影像**: 顯示經過處理的影像
- **量測結果**: 顯示計算出的面積、周長、深度、體積
- **最近量測記錄**: 顯示歷史量測記錄

## 影像品質要求

為了獲得最佳的分析結果，建議影像符合以下要求：

### 基本要求
- **解析度**: 建議 640x480 以上
- **格式**: 清晰的傷口影像
- **光線**: 充足且均勻的照明
- **角度**: 垂直拍攝，避免過度傾斜

### 品質評估
系統會自動評估影像品質，包括：
- **亮度分析**: 檢查影像是否過暗或過亮
- **對比度**: 確保傷口區域與背景有足夠對比
- **清晰度**: 檢測影像是否模糊

## 注意事項

### 檔案大小
- 建議影像檔案大小不超過 10MB
- 過大的檔案可能影響載入速度

### 影像內容
- 確保影像中傷口區域清晰可見
- 避免過多的背景雜物
- 建議包含比例尺或參考物件

### 處理時間
- 影像處理時間取決於影像大小和複雜度
- 高解析度影像需要更長的處理時間

## 故障排除

### 常見問題

**Q: 無法載入影像檔案**
A: 檢查檔案格式是否支援，檔案是否損壞

**Q: 載入後影像顯示異常**
A: 檢查影像檔案是否為標準格式，嘗試重新載入

**Q: 處理結果不準確**
A: 檢查影像品質，確保傷口區域清晰可見

**Q: 處理時間過長**
A: 嘗試使用較小解析度的影像，或檢查系統資源

### 錯誤訊息

- **"影像載入失敗"**: 檔案格式不支援或檔案損壞
- **"請先載入影像"**: 需要先載入影像才能進行處理
- **"影像處理失敗"**: 處理過程中發生錯誤，檢查日誌檔案

## 測試影像

系統包含測試影像生成器 (`test_image_generator.py`)，可以生成模擬的傷口影像用於測試：

```bash
# 執行測試影像生成器
python test_image_generator.py
```

這會生成多個不同解析度的測試影像，可用於驗證載入本地影像功能。
