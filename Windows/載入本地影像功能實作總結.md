### 2. 影像品質評估

```csharp
private double CalculateImageQuality(Bitmap bitmap)
{
    // 計算平均亮度
    double totalBrightness = 0;
    for (int x = 0; x < bitmap.Width; x += 10)
    {
        for (int y = 0; y < bitmap.Height; y += 10)
        {
            var pixel = bitmap.GetPixel(x, y);
            totalBrightness += (pixel.R + pixel.G + pixel.B) / 3.0;
        }
    }
    
    var avgBrightness = totalBrightness / ((bitmap.Width / 10) * (bitmap.Height / 10));
    return Math.Min(100, avgBrightness * 100 / 255);
}
```

### 3. 格式轉換

```csharp
private ImageData ConvertBitmapSourceToImageData(BitmapSource bitmapSource)
{
    // 將 BitmapSource 轉換為 Bitmap
    var width = bitmapSource.PixelWidth;
    var height = bitmapSource.PixelHeight;
    var stride = width * ((bitmapSource.Format.BitsPerPixel + 7) / 8);
    var buffer = new byte[height * stride];
    
    bitmapSource.CopyPixels(buffer, stride, 0);
    
    var bitmap = new Bitmap(width, height, System.Drawing.Imaging.PixelFormat.Format32bppArgb);
    var bitmapData = bitmap.LockBits(new Rectangle(0, 0, width, height), 
        System.Drawing.Imaging.ImageLockMode.WriteOnly, 
        System.Drawing.Imaging.PixelFormat.Format32bppArgb);
    
    System.Runtime.InteropServices.Marshal.Copy(buffer, 0, bitmapData.Scan0, buffer.Length);
    bitmap.UnlockBits(bitmapData);
    
    return new ImageData
    {
        RgbImage = bitmap,
        Width = width,
        Height = height,
        Timestamp = DateTime.Now,
        QualityScore = CalculateImageQuality(bitmap)
    };
}
```

## 使用指南

### 1. 基本操作

1. **啟動應用程式**
2. **點擊「載入本地影像」**
3. **選擇影像檔案**
4. **點擊「處理載入影像」**
5. **查看結果**

### 2. 注意事項

- 建議影像解析度 640x480 以上
- 檔案大小不超過 10MB
- 確保傷口區域清晰可見
- 避免過度傾斜的拍攝角度
