# iPhone 傷口自動化測量程式

基於iPhone相機和LiDAR感測器的傷口自動化測量與分類系統，整合ImageJ影像處理技術。

## 系統架構

### 主要模組
1. **Capture Module** - 相機拍攝與LiDAR深度資料獲取
2. **Pre-processing Module** - 影像標準化與品質判定
3. **QA & Filter Module** - 品質檢查與初步分類
4. **ImageJ-Core** - 傷口分割與測量計算
5. **Classification Module** - 詳細傷口分類與風險評估
6. **Result UI & Local DB** - 結果展示與資料儲存

### 技術特點
- 支援iPhone Pro系列LiDAR深度感測
- 整合CoreML機器學習模型
- ImageJ演算法的Swift實現
- CoreData本地資料庫
- SwiftUI現代化介面

## 功能特色

### 影像品質控制
- 自動白平衡校正（Gray World演算法）
- 透視校正與ROI自動偵測
- SNR ≥ 20dB品質門檻
- 運動模糊檢測（Laplacian variance）

### 測量功能
- **面積測量**：基於輪廓偵測的精確面積計算
- **體積測量**：結合深度資料的3D體積估算
- **組織分析**：壞死/肉芽/上皮組織比例分析

### 分類功能
- **急性/慢性分類**：基於顏色和紋理特徵
- **感染風險評估**：多維度風險指標分析
- **癒合階段判定**：發炎期/增生期/成熟期識別

## 技術參數

### 相機設定
- ISO: 100-400
- 快門速度: 1/60-1/125秒
- 自動白平衡校正

### LiDAR設定
- 深度解析度: 256×192 @ 30fps
- 信心度門檻: ≥ 0.7
- 深度覆蓋率: ≥ 80%

### 品質門檻
- SNR: ≥ 20 dB
- 模糊度: Variance(Laplacian) ≥ 100
- 分類信心度: ≥ 0.7

## 使用說明

1. **開始測量**：點擊「開始測量」按鈕
2. **拍攝影像**：對準傷口拍攝，確保良好的光線和角度
3. **自動處理**：系統自動進行影像處理和分析
4. **查看結果**：檢視測量數據、分類結果和建議
5. **儲存紀錄**：資料自動儲存至本地資料庫

## AI模型整合

程式支援CoreML模型整合：
1. 將訓練好的`.mlmodel`轉換為`.mlmodelc`
2. 新增至專案資源
3. 模組會自動載入並使用AI模型進行分類

若無AI模型，系統會使用基於特徵的傳統分類方法。

## 注意事項

### 拍攝建議
- 確保充足且均勻的光線
- 避免陰影和反光
- 保持手機穩定，避免模糊
- 傷口應完整出現在畫面中

### 精度限制
- 測量精度受拍攝距離和角度影響
- LiDAR深度資料在極近距離可能不準確
- 建議多次測量取平均值
