# 方形校正貼紙設計規格 v2.0

## 🎯 設計概述

全新設計的20mm×20mm方形校正貼紙，整合色彩校正、尺寸校正和3D透視校正功能。

## 📐 整體規格

- **外形**: 20mm × 20mm 正方形
- **材質**: 啞光白色底材，防反光處理
- **印刷**: 高精度數位印刷，色彩標準化
- **厚度**: 0.1mm，附背膠，易撕除

## 🎨 色彩校正點配置

### 主色彩點配置（上下左右四方位）
```
       [R] 紅色
        │
[B]藍色 ── ○ ── [Y]黃色  
        │     
       [G] 綠色
```

#### 精確位置設計
- **紅色點 (R)**: 位於上方中心，距邊緣3mm
  - 色值: #FF0000 (標準紅)
  - 直徑: Ø3mm
  - 位置: (10mm, 3mm)

- **黃色點 (Y)**: 位於右方中心，距邊緣3mm  
  - 色值: #FFFF00 (標準黃)
  - 直径: Ø3mm
  - 位置: (17mm, 10mm)

- **綠色點 (G)**: 位於下方中心，距邊緣3mm
  - 色值: #00FF00 (標準綠)
  - 直徑: Ø3mm  
  - 位置: (10mm, 17mm)

- **藍色點 (B)**: 位於左方中心，距邊緣3mm
  - 色值: #0000FF (標準藍)
  - 直徑: Ø3mm
  - 位置: (3mm, 10mm)

### 中心灰度參考點
- **18%標準灰**: 位於中心
  - 色值: #2D2D2D (18% 反射率)
  - 直徑: Ø4mm
  - 位置: (10mm, 10mm)

## 🎯 功能整合設計

### 1. 色彩校正功能
- **RGBY四色標準**: 覆蓋完整色彩空間
- **白平衡校正**: 使用18%灰度參考  
- **色溫調整**: 基於標準色彩計算
- **伽瑪校正**: 參考灰階漸變

### 2. 尺寸校正功能
- **精確尺寸**: 20mm×20mm標準正方形
- **像素比例**: 自動計算cm/pixel
- **面積校正**: 400mm²標準面積

### 3. 透視校正功能
- **四角定位**: 凸點提供精確角點
- **平面檢測**: 凸點陰影分析角度偏差
- **透視變換**: 自動計算變換矩陣

## 🔍 檢測算法設計

### 色彩點檢測
```python
# HSV色彩空間檢測範圍
COLOR_RANGES = {
    'red': [(0, 100, 100), (10, 255, 255), (170, 100, 100), (180, 255, 255)],
    'yellow': [(20, 100, 100), (30, 255, 255)],  
    'green': [(40, 100, 100), (80, 255, 255)],
    'blue': [(100, 100, 100), (130, 255, 255)],
    'gray': [(0, 0, 40), (180, 30, 80)]  # 18%灰
}
```

### 凸點檢測
- **邊緣檢測**: Canny算子找出凸點輪廓
- **圓形檢測**: HoughCircles找出Ø2mm圓形
- **陰影分析**: 計算凸點陰影方向和長度
- **深度估算**: 基於陰影推算拍攝角度

## 🎨 印刷色彩標準

### ICC色彩配置檔
```
Red:   sRGB(255, 0, 0)   → Lab(53.2, 80.1, 67.2)
Yellow: sRGB(255, 255, 0) → Lab(97.1, -21.6, 94.5)  
Green: sRGB(0, 255, 0)   → Lab(87.7, -86.2, 83.2)
Blue:  sRGB(0, 0, 255)   → Lab(32.3, 79.2, -107.9)
Gray:  sRGB(45, 45, 45)  → Lab(20.5, 0.0, 0.0)
```

### 印刷品質控制

- **位置精度**: ±0.1mm
