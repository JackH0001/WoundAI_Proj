# iPhone 傷口自動化測量程式 - 開發文檔

## 專案概述
這是一個基於iPhone相機和LiDAR感測器的傷口自動化測量與分類系統，整合了ImageJ影像處理技術和機器學習分類功能。

## 系統架構

### 核心模組
1. **CaptureModule** - 負責相機拍攝和LiDAR深度資料獲取
2. **PreProcessingModule** - 執行影像標準化和品質判定
3. **QAFilterModule** - 進行品質檢查和初步分類
4. **ImageJCore** - 實現ImageJ的傷口分割和測量功能
5. **ClassificationModule** - 執行詳細的傷口分類和風險評估
6. **WoundRecord + CoreData** - 處理資料儲存和歷史紀錄

### 資料流程
```
相機拍攝 → 預處理 → 品質檢查 → ImageJ分析 → 分類評估 → 結果顯示 → 資料儲存
```

## 技術特點

### 影像處理
- Gray World白平衡校正
- LiDAR輔助的透視校正
- 自動ROI偵測
- SNR和模糊度品質檢查

### 測量功能
- 基於輪廓的面積計算
- 結合深度資料的體積估算
- 組織類型分析（壞死/肉芽/上皮）

### 分類系統
- 支援CoreML模型整合
- 基於特徵的備選分類
- 多維度風險評估
- 癒合階段識別

## 重要技術參數

### 相機設定
- ISO: 100-400
- 快門: 1/60-1/125秒
- 高解析度拍攝啟用

### 品質門檻
- SNR: ≥ 20 dB
- 深度覆蓋: ≥ 80%
- 分類信心度: ≥ 0.7

### LiDAR參數
- 深度解析度: 256×192
- 幀率: 30fps
- 信心度門檻: 0.7

## CoreML模型整合

### 支援的模型
1. `WoundClassification.mlmodelc` - 初步分類
2. `WoundDetailedClassification.mlmodelc` - 詳細分類

### 模型輸入
- 影像尺寸: 256×256 (RGB)
- 可選深度通道: 256×256 (Depth)

### 模型輸出
- 急性/慢性機率
- 感染風險分數
- 癒合階段分類

## 測試與除錯

### 模擬器限制
- 無法使用相機和LiDAR
- 可使用範例資料測試UI

### 實機測試
- 需要iPhone 12 Pro+
- 確保相機和ARKit權限
- 在不同光線條件下測試

### 常見問題
1. **LiDAR資料無效** - 檢查ARKit設定和裝置支援
2. **分類結果不準確** - 調整品質門檻或模型參數
3. **記憶體問題** - 處理大影像時注意記憶體管理
