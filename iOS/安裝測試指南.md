# iPhone 傷口自動化測量程式 - 安裝測試指南

## 📱 程式概述

本程式是基於iPhone相機和LiDAR感測器的傷口自動化測量與分類系統，整合ImageJ影像處理技術和機器學習分類功能，可精確測量傷口面積、體積，並提供智能分類和治療建議。

---

## 🛠️ 系統需求

### 硬體需求
- **iPhone**: iPhone 12 Pro、iPhone 12 Pro Max、iPhone 13 Pro、iPhone 13 Pro Max、iPhone 14 Pro、iPhone 14 Pro Max、iPhone 15 Pro、iPhone 15 Pro Max（必須具備LiDAR感測器）
- **iOS版本**: iOS 17.0或更新版本
- **儲存空間**: 至少100MB可用空間

### 開發環境需求
- **macOS**: macOS 13.0 (Ventura) 或更新版本
- **Xcode**: Xcode 15.0或更新版本
- **Apple開發者帳號**: 用於裝置測試和App Store發布

## 🔧 安裝步驟

### 第一步：準備開發環境

1. **安裝Xcode**
   - 從Mac App Store下載並安裝Xcode 15.0+
   - 確保已安裝iOS 17.0 SDK

2. **設定Apple開發者帳號**
   - 登入Xcode → Preferences → Accounts
   - 新增您的Apple ID（免費開發者帳號即可用於測試）

### 第二步：開啟專案

1. **方法一：終端機開啟**
   ```bash
   cd "C:\Users\jack_\.cursor-tutor\傷口自動化量測分類"
   open WoundMeasurementApp.xcodeproj
   ```

2. **方法二：Xcode開啟**
   - 啟動Xcode
   - 選擇 "Open a project or file"
   - 導航到專案資料夾
   - 選擇 `WoundMeasurementApp.xcodeproj`

### 第三步：專案設定

1. **選擇專案設定**
   - 點擊專案根目錄 (WoundMeasurementApp)
   - 選擇 "WoundMeasurementApp" target

2. **設定簽名與功能**
   - 前往 "Signing & Capabilities" 頁籤
   - **Team**: 選擇您的Apple開發者帳號
   - **Bundle Identifier**: 修改為唯一ID（例如：`com.yourname.woundmeasurement`）
   - **iOS Deployment Target**: 確認為17.0或更高

3. **檢查必要權限**（已預設包含）
   - Camera Usage Description
   - Photo Library Usage Description
   - ARKit support

## 📊 使用說明

### 主要功能

#### 1. 傷口測量
- **面積測量**：自動計算傷口面積（cm²）
- **體積測量**：結合深度資料計算體積（cm³）
- **周長測量**：精確測量傷口邊界長度
- **深度分析**：LiDAR提供深度資訊

#### 2. 智能分類
- **急性/慢性分類**：基於影像特徵判定傷口類型
- **感染風險評估**：多維度分析感染可能性
- **癒合階段識別**：判定發炎期/增生期/成熟期
- **組織成分分析**：壞死/肉芽/上皮組織比例

#### 3. 品質控制
- **自動品質檢查**：SNR≥20dB標準
- **模糊度檢測**：確保影像清晰度
- **深度覆蓋驗證**：LiDAR資料完整性≥80%

### 操作流程

#### 拍攝準備
1. **環境準備**
   - 確保充足且均勻的光線
   - 避免強烈陰影和反光
   - 保持手機穩定

2. **拍攝技巧**
   - 距離傷口15-30cm
   - 保持手機平行於傷口表面
   - 確保傷口完整出現在畫面中
   - 避免手部抖動

#### 測量步驟
1. **開始測量**
   - 點擊主畫面「開始測量」按鈕
   - 相機介面自動開啟

2. **拍攝影像**
   - 對準傷口調整角度和距離
   - 點擊「拍攝」按鈕
   - 等待拍攝完成

3. **自動處理**
   - 系統自動進行影像處理
   - 品質檢查和標準化
   - 傷口分割和測量計算
   - 分類分析和風險評估

4. **查看結果**
   - 檢視測量數據
   - 了解分類結果
   - 閱讀治療建議
   - 儲存測量紀錄

#### 歷史管理
1. **查看紀錄**
   - 點擊「查看歷史紀錄」
   - 瀏覽過往測量結果
   - 比較不同時間點的數據

2. **詳細分析**
   - 點擊任一紀錄查看詳情
   - 檢視完整測量報告
   - 了解組織組成分析

## ⚠️ 常見問題與解決方案

### 編譯問題

**問題1**: "Signing for WoundMeasurementApp requires a development team"
```
解決方案：
1. 前往 Signing & Capabilities
2. 在 Team 下拉選單選擇您的Apple開發者帳號
3. 確認Bundle Identifier是唯一的
```

**問題2**: "No such module 'ARKit'"
```
解決方案：
1. 確認部署target為iOS 17.0+
2. 檢查 ARKit framework 是否正確連結
3. 在需要的檔案頂部加入 import ARKit
```

### 執行問題

**問題3**: 相機無法啟動
```
解決方案：
1. 檢查 Info.plist 中的相機權限描述
2. 確認在實機上測試（模擬器不支援相機）
3. 在設定中手動授予相機權限
```

**問題4**: LiDAR資料無效
```
解決方案：
1. 確認使用支援LiDAR的iPhone機型
2. 檢查ARKit配置是否正確
3. 確保拍攝距離適中（15-30cm）
```

### 性能問題

**問題5**: 處理速度過慢
```
解決方案：
1. 檢查影像解析度設定
2. 確認在背景執行緒處理影像
3. 使用Instruments檢查性能瓶頸
```

**問題6**: 記憶體使用過高
```
解決方案：
1. 及時釋放不需要的影像資料
2. 使用autorelease pool處理大量資料
3. 監控記憶體使用量
```
