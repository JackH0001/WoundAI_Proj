# WoundAI 臨床前瞻驗證計畫（初步 v1）

目的：以前瞻收案，驗證 2D 自動量測（面積）、組織分型、PUSH 嚴重度之準確度與一致性，作為 TFDA 送審臨床評估依據。輔助、非診斷；GT 須醫師驗證。

## 1. 設計
- 前瞻、單/多中心、觀察性；同一傷口由 App（含校正貼紙）拍攝，與參考方法同時取得。
- 對照（面積 GT）：透明描圖+方格紙或專業傷口尺規 / 校正後數位平面測量；組織/PUSH GT：≥2 位傷口照護醫師獨立判讀，歧異第三方仲裁。

## 2. 收案
- 納入：≥20 歲、慢性傷口（壓瘡/糖尿病足/靜脈/動脈/術後/燒傷）、可貼校正貼紙、簽署知情同意。
- 排除：無法去識別、拒絕同意、傷口位置無法平貼貼紙。
- 多元化：部位/類型/膚色/光線分層，避免單一分布偏差（呼應現況 n=5 樣本小、偏樂觀）。

## 3. 樣本數（初估）
- 面積一致性（Bland–Altman/ICC）：以 ICC≥0.90、半寬精度 0.05、α0.05、power0.8 → 約 **85–120 傷口**；考量退出加 15% → **~130**。
- 組織分型：每類（肉芽/腐肉/壞死/上皮）≥50 例以估敏感度/特異度（與飛輪門檻一致）。
- 傷口類型分類：每類≥? 視未來啟動而定（現階段不宣稱）。

## 4. 主要/次要指標
- **主要**：面積 自動 vs 參考 → Bland–Altman 偏差與 95% LoA、ICC、MAPE（目標 ≤ ~10–15% 實務級；現乾淨貼紙模擬 ~2.7%）。
- **次要**：組織分型 vs 醫師共識（per-class 敏感度/特異度/κ）；PUSH 自動 vs 醫師（ICC、加權κ）；雙軌路由難例正確上雲率；拍攝品質把關有效性（NG 重拍率）。
- 信效度：判讀者間信度（醫師 κ）、重測信度（同傷口多次）。

## 5. 統計
- 連續量（面積/PUSH）：Bland–Altman、ICC(2,1)、MAPE；分類：混淆矩陣、敏感度/特異度/κ。
- 子群分析：部位/類型/膚色/光線/角度；異常值與失敗模式記錄（貼紙未偵測、斜角、模糊）。

## 6. 倫理與資料
- IRB/REC 審查；知情同意（雙層：照護/去識別化訓練，可撤回；電子簽名）。
- 去識別化後方上傳；稽核日誌；資料保存與銷毀政策；個資法/醫療法遵循。

## 7. 與工程驗證銜接
- 臨床 GT（醫師雙閱）回流飛輪（correction_iou 排序）→ 再訓練（回歸守門 ≥0.88、無類別下降）→ 版本治理。
- 三端計分一致（金標差異 0）為前提；App 端到端對齊後端 `/api/v1/*`。

## 8. 里程碑
1. IRB 送審 + 貼紙量產 + App 收案版鎖定。 2. 試收案(~10)校準流程。 3. 正式收案至樣本數。 4. 統計分析+臨床評估報告 → TFDA。

## 9. 限制
現為計畫；實際門檻/樣本數待生統與臨床確認。不宣稱醫療級精確度直到前瞻達標。
