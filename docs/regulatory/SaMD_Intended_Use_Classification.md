# WoundAI 2D — SaMD 用途宣告與分級（模板，鏡像 WoundAI3D Z3/Z6）
> 狀態：草案模板,待法規顧問確認。對應 IMDRF SaMD 框架 / TFDA / FDA。

## 1. 預期用途 Intended Use
WoundAI 2D 為**傷口 2D 影像量測與記錄之輔助工具**:以手機相機＋校正貼紙量測傷口**面積**並追蹤變化,提供**規則式輔助提示**(非診斷)與醫師協作標註。**不單獨用於診斷或治療決策**。
- 使用者:醫護人員(臨床)/ 受訓照護者。
- 病患族群:慢性傷口(壓瘡、糖尿病足等)。
- 量測標的:面積(cm²);（選配,研究級）LiDAR 體積。

## 2. SaMD 分類（IMDRF）
- 醫療情境重要性:慢性傷口管理(非危急)。
- 資訊用途:**驅動臨床管理(輔助)**,非「診斷/治療直接決定」。
- 初判 SaMD Category：**II（待顧問確認）**;AI 診斷功能未啟用前維持「量測/記錄」低風險定位。

## 3. 宣稱邊界（誠實,對應現況）
| 功能 | 可宣稱 | 不可宣稱 |
|---|---|---|
| 面積量測 | phantom 實測 ~3–5%(方形20mm+面積比例法) | 臨床級精度(待 M3 臨床前驗證) |
| 分割 | 輔助初稿 | 自動診斷 |
| 分類/嚴重度 | **規則式提示** | **AI 診斷**(模型未落地) |
| LiDAR 體積 | 研究/輔助(feature flag) | 醫療級體積 |
