# WoundAI 2D — 風險管理檔（ISO 14971 模板,鏡像 WoundAI3D RiskRegister/Z5）
> 草案。每條:危害→可預見事件序列→傷害→嚴重度×可能性→風險控制→剩餘風險。

## 危害分析（核心條目,待擴充）
| ID | 危害 | 事件序列 | 潛在傷害 | 控制措施 | 剩餘風險 |
|---|---|---|---|---|---|
| H1 | 面積**低估** | 斜拍/曲面/校正貼紙遠離→透視低估 | 低估傷口惡化、延誤處置 | 貼紙緊貼傷口+面積比例法;輸出不確定度;斜拍警示 | 中→低 |
| H2 | 面積**高估** | 分割過切(抓到周邊皮膚/紅色干擾) | 誤判惡化、過度處置 | mask_refine 去框邊;集成模型;醫師修邊覆核 | 中→低 |
| H3 | **前處理錯誤** | 各端前處理不一致/用錯(如 wsm 曾誤用 RGB) | 全面錯遮罩→系統性誤量測 | **SSOT 單一來源+codegen+CI 一致性測試**(已落地) | 高→低 |
| H4 | 誤認「規則式提示」為診斷 | UI 未清楚標示 | 過度信任、延誤就醫 | UI 明標「輔助、非診斷、需醫師確認」;宣稱邊界控管 | 中→低 |
| H5 | LiDAR 體積誤差被當醫療級 | 研究級數值未標示 | 誤判深度/體積 | feature flag+「研究級」標示+不確定度+與2D分軌 | 中→低 |
| H6 | PHI 外洩 | 上傳/標註未加密、密碼硬編碼 | 隱私侵害 | 去識別、加密、移除硬編碼密碼、稽核軌跡 | 高→低 |
