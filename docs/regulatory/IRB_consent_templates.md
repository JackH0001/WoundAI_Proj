## 一、受試者說明暨同意書（Informed Consent Form）

**研究名稱**：WoundAI 二維自動化傷口量測與嚴重度輔助評估之前瞻驗證
**主持人/機構**：〔PI〕/〔醫療機構〕　**IRB 編號**：〔__〕　**版本/日期**：v1 / 〔__〕

### 1. 研究目的
驗證以行動裝置加校正貼紙拍攝慢性傷口，自動量測**面積**、推估**組織分型**與 **PUSH 嚴重度** 之準確度與一致性，作為臨床輔助工具評估。**本工具為輔助參考、非診斷**，不取代醫師判斷。

### 2. 研究程序
經同意後，於常規換藥/門診時，由醫護以本 App（傷口旁貼校正貼紙）拍攝傷口影像；同時以參考方法（描圖/尺規或數位平面測量）取得對照。影像之傷口邊界與組織由 ≥2 位醫師判讀為比對標準。不增加額外侵入性處置。

### 3. 風險與不便
極低：僅多一次拍照與貼紙黏貼（可能輕微皮膚不適）。影像含傷口；個資保護見第 6 點。

### 4. 預期效益
個人未必直接受益；研究有助改善傷口照護量測工具與後續病患。

### 5. 雙層知情同意（可分別勾選，得隨時撤回）
- ☐ **①（必填）照護用途**：同意拍攝/儲存傷口影像作為本人照護記錄與量測。
- ☐ **②（選填）去識別化貢獻 AI 訓練**：同意將**去識別化後**之影像/標註用於模型訓練與改良；**得隨時撤回**，撤回後對應資料下架並排除後續訓練，且不影響您的照護。
> 未同意①不進行拍攝；未同意②不上傳訓練平台。

### 6. 個資保護與去識別化
可識別資訊（姓名/病歷號）僅存醫療端供照護；上傳訓練平台前**一律去識別化**（移除 EXIF/定位/裝置資訊、遮蔽畫面外可識別物、病歷號改不可逆雜湊、僅裁切傷口+貼紙區）。所有存取/上傳/訓練留**稽核紀錄**。臨床病程資料屬醫療紀錄系統，另依規管理。

### 7. 自願參與與退出
完全自願；隨時可退出或撤回②，不影響權益與既有醫療。

## 二、IRB 送審摘要（Study Synopsis）
| 項目 | 內容 |
|---|---|
| 名稱 | WoundAI 2D 自動量測/分型/PUSH 前瞻驗證 |
| 設計 | 前瞻、觀察性、單/多中心 |
| 對象 | ≥20 歲慢性傷口患者(壓瘡/糖足/靜脈/動脈/術後/燒傷)，可貼貼紙、簽同意 |
| 樣本數(初估) | 面積一致性 ~130 傷口;組織每類≥50(詳見前瞻驗證計畫) |

| 次要 | 組織分型(κ)、PUSH(ICC/加權κ)、雙軌上雲率、品質把關 |
| 風險分級 | 最小風險(非侵入、非診斷、人為最終判讀) |
| 倫理 | 知情同意(雙層可撤回)、去識別化、稽核、資料保存/銷毀 |

## 三、資料管理與去識別化計畫（Data Management Plan）
1. **蒐集**：傷口影像(+選配深度)、量測/標註中繼資料、雙層同意與電子簽名紀錄。
2. **去識別化**(上傳前)：EXIF/GPS/裝置移除、外圍個資遮蔽、病歷號→不可逆雜湊、裁切傷口+貼紙。
3. **儲存/存取**：去識別資料存訓練平台(無姓名/病歷號，以代碼管理)；最小權限；全程稽核日誌。
4. **保存/銷毀**：依機構與法規期限保存，逾期或撤回即下架/銷毀並記錄。
5. **撤回處理**：撤回②→對應去識別資料移出再訓練佇列、排除後續訓練；已納入已發布模型者於下輪重訓不再納入。
6. **外洩應變**：依機構資安事件流程通報與處置。
